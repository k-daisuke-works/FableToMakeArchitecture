REQUIREMENTS — エージェント生成 要件定義フォーム

どんなエージェント体制が欲しいか教えてください

必要な項目を埋めて「指示文を生成」を押すと、Claude Code にそのまま貼り付けられる /design-agents の指示文ができます。分からない項目は空欄のままで構いません (エージェント側が推測し、重大なものだけ確認してきます)。

1. プロジェクト名
この体制の名前。生成先フォルダ名(projects/〜)の元になります。日本語でも英語でも可。未定なら空欄でOK(目的から命名されます)。
2. 目的 必須
このエージェント体制に何をさせたいか。自由に書いてください。(例: Spring Boot で社内向け勤怠管理APIを開発したい / 競合他社の動向を毎週調査してレポートさせたい)
3. 体制の種類
4. 技術スタック・対象環境
言語・フレームワーク・対象システムなど。未定なら空欄でOK。
5. 規模感
6. リスク要素(あてはまるもの全て)
7. 品質要求
8. 繰り返し発生しそうな作業
週1回以上やりそうな定型作業。スキル化の候補になります。(例: 画面の追加、週次レポート作成、リリース前チェック)
9. 人間(あなた)の関与度
10. その他の要望
こういうエージェントが欲しい、これはやらせたくない、参考にしたい既存体制、など何でも。
「2. 目的」は必須です。何をさせたい体制なのか記入してください。

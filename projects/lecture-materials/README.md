# lecture-materials — 講義資料作成エージェント環境

医療・福祉分野の非エンジニア従事者向け「生成AIの基礎知識」講義の資料一式
(講義構成・スライド・台本)を、テーマと聴衆を伝えるだけで自律生成する環境です。

## 最初の一歩

**`tools/request-form.html` をブラウザで開いてください。** テーマ・聴衆・時間などの
項目を埋めて「指示文を生成」を押すと、そのまま Claude Code に貼り付けられる指示文
(要件定義書)ができます(.md ファイルとして保存して、そのパスを渡してもOK)。

フォームを使わず、直接指示することもできます:

```
/new-lecture テーマ: 生成AIの基礎知識、聴衆: 介護施設の職員(ITに不慣れ)、時間: 60分、講義日: 8/20
```

構成案 → 最新動向の確認 → スライド → 台本 → 公開前チェックまで自律で完走し、
最後に公開可否の確認を求めてきます。成果物は `lectures/20260820-genai-basics/` のような
ディレクトリに `outline.md`(構成)・`slides.md`(Marp スライド)・`script.md`(台本)として
生成されます。

## スキル一覧

| スキル | いつ使うか |
|---|---|
| `/new-lecture` | 新しい講義の資料一式を作る(テーマ・聴衆・時間を指定) |
| `/update-lecture` | 既存資料を最新の生成AI動向に更新する |
| `/adjust-audience` | 聴衆・時間・トーンを変えた派生版を作る(元は上書きしない) |
| `/publish-check` | 公開・配布前のチェック。他スキルの最後にも自動実行される |

**公開前は必ず人間の最終確認が必要です。** `/publish-check` は content-reviewer
サブエージェント(独立ファクトチェッカー)の検証と修正まで自動で行いますが、
公開可否の判断はあなたが行います。

## マスター構成テンプレートを育てる

`masters/base-outline.md` が毎回の講義の下敷きになる定番構成です。まずこれを開いて、
自分の講義スタイル(定番の章立て・時間配分・免責文)に合わせて編集してください。
ここを育てるほど毎回の生成が自分好みになります。講義ごとの資料はこのファイルを
変更しません(更新したいときは `/update-lecture` で masters を指定)。

## スライドの PDF / pptx 出力

スライドは Marp 形式の Markdown で管理します。エクスポートには Node.js が必要です:

```bash
npx @marp-team/marp-cli lectures/<日付-テーマ>/slides.md -o slides.pdf   # PDF(標準)
npx @marp-team/marp-cli lectures/<日付-テーマ>/slides.md -o slides.pptx  # PowerPoint
```

pptx 直接作成などに形式を変えたい場合は、`CLAUDE.md` の「成果物の形式」節だけを
書き換えれば全スキルに反映されます。

## 既存リポジトリで使うには

資料を別のリポジトリで管理したい場合は、以下をコピーしてください:

```bash
cp -r .claude/ CLAUDE.md masters/ tools/ <あなたのリポジトリ>/
mkdir -p <あなたのリポジトリ>/lectures
```

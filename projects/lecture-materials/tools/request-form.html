REQUEST — 講義資料作成 要件定義フォーム

どんな講義資料が欲しいか教えてください

項目を埋めて「指示文を生成」を押すと、このプロジェクトで起動した Claude Code に そのまま貼り付けられる指示文ができます。分からない項目は空欄のままで構いません (エージェント側が推測し、重大なものだけ確認してきます)。 生成した指示文はコピーのほか、ファイル(.md)として保存もできます。

依頼の種類
選んだ種類に関係する項目だけが下に表示されます。
テーマ 必須
何についての講義か。(例: 生成AIの基礎知識 / 生成AIと個人情報の付き合い方)
講義日
開催日。成果物フォルダ名(lectures/日付-テーマ)に使われます。未定なら空欄でOK。
講義時間
持ち時間。時間配分の計算に使われます。(例: 60分 / 90分 / 質疑込み45分)
聴衆(職種・前提知識)
誰に向けた講義か。職種・人数・ITへの馴染み度など。用語の平易さと例え話の選び方に使われます。
トーン・難易度の要望
話し方や深さの希望があれば。(例: とにかく易しく / 管理職向けにリスク管理を厚めに)
対象の資料
元にする資料の場所。lectures/ のフォルダ名(例: 20260820-genai-basics)、またはマスター構成を更新するなら masters/base-outline.md。未指定なら最新の講義が対象になります。
重点を置きたい内容・追加したいトピック
今回特に扱いたいこと、前回の質疑で出た話題など。(例: 音声入力の活用 / 施設での禁止事項の作り方)
その他の要望
上記に収まらないことは何でも。(例: 配布資料も想定してスライドは文字多めに)
「テーマ」は必須です。何についての講義か記入してください。

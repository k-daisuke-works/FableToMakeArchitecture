# stock-investment — 株式運用リサーチエージェント環境

元手資金と運用方針を伝えると、Web 調査に基づいて具体的な銘柄・株数・目安金額・
タイミングまで落とし込んだ提案書を作る個人向けリサーチ環境です。

> ⚠️ この環境が出すのは**参考情報**であり、投資助言ではありません。発注の実行や
> 証券会社との連携は行いません。株価等の数値はリアルタイムではなく取得時点のものです。
> 投資の最終判断はご自身の責任で行ってください。

## 最初の一歩

**`tools/request-form.html` をブラウザで開いてください。** 依頼の種類(提案書作成/
銘柄スクリーニング/保有の見直し)を選んで項目を埋め、「指示文を生成」を押すと、
そのまま Claude Code に貼り付けられる指示文ができます(.md 保存してパスを渡してもOK)。

フォームを使わず直接指示することもできます:

```
/portfolio-proposal 元手100万円。日本株中心、長期でコツコツ資産形成したい。リスクは低め。
```

## 仕事の流れと2つの節目

```
/portfolio-proposal → 運用方針の整理 → 【節目① 方針の確定】
                   → stock-researcher が銘柄を Web 調査(出典・取得日時つき)
                   → 提案書を reports/proposals/ に出力 → 【節目② 提案の提示】
```

節目以外は自律で進みます。すべてのレポートは「冒頭に免責文言/全数値に出典URL+
取得日時/事実と推測の区別」の3ルールを満たします。

## スキルとエージェント

| 構成物 | 役割 | モデル |
|---|---|---|
| `/portfolio-proposal` | 方針確定→調査→銘柄・株数・金額・タイミングの提案書 | (メイン) |
| `/stock-screening` | 条件指定で候補銘柄を絞り込む(単発利用) | (メイン) |
| `/portfolio-review` | 保有銘柄の見直し・乗り換え判断案 | (メイン) |
| `stock-researcher` | 銘柄・市場の Web 調査。要約と出典だけ返す(読み取り専用) | sonnet |

調査をサブエージェントに分離しているのは、検索結果・ページ本文でメイン会話の
コンテキストを消費しないため(トークン効率)です。

## 成果物の置き場所

- 提案書: `reports/proposals/<日付>-<方針名>.md`
- スクリーニング結果: `reports/screening/<日付>-<条件名>.md`
- 見直しレポート: `reports/reviews/<日付>-review.md`

## 既存リポジトリで使うには

```bash
cp -r .claude/ CLAUDE.md tools/ <あなたのリポジトリ>/
mkdir -p <あなたのリポジトリ>/reports
```

REQUEST — 株式運用リサーチ 要件定義フォーム

どんな運用をしたいか教えてください

項目を埋めて「指示文を生成」を押すと、このプロジェクトで起動した Claude Code に そのまま貼り付けられる指示文ができます。分からない項目は空欄のままで構いません (エージェント側が推測し、節目で確認してきます)。生成した指示文はコピーのほか、 ファイル(.md)として保存もできます。

このエージェントが出すのは参考情報であり、投資助言ではありません。発注の実行は 行わず、株価等の数値は取得時点のものです。投資の最終判断はご自身の責任で行ってください。

依頼の種類
選んだ種類に関係する項目だけが下に表示されます。
元手資金 必須
運用に回せる金額。通貨も書いてください。
対象市場 必須
あてはまるもの全て。
運用期間
運用目的
何のための運用か。(例: 老後の資産形成 / 配当収入がほしい / 値上がり益を狙いたい)
リスク許容度
制約・希望
除外したい業種、応援したい分野、1銘柄あたりの上限など何でも。
スクリーニング条件 必須
絞り込みの条件。(例: 配当利回り3%以上 / 時価総額1000億円以上 / IT業種)
抽出件数
上位何件まで出すか。未指定なら10件。
保有状況 必須
銘柄・株数・取得単価を1行ずつ。
運用方針の変更点
前回から方針に変化があれば。(例: 現金比率を増やしたい)
見直しの観点
特に見てほしいこと。(例: 損切りすべきか / もっと良い乗り換え先があるか)

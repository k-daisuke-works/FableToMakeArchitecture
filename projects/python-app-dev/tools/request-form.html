何を作りたいか教えてください

項目を埋めて「指示文を生成」を押すと、このプロジェクトで起動した Claude Code に そのまま貼り付けられる指示文ができます。分からない項目は空欄のままで構いません (エージェント側が推測し、節目で確認してきます)。生成した指示文はコピーのほか、 ファイル(.md)として保存もできます。

依頼の種類
選んだ種類に関係する項目だけが下に表示されます。どちらも実装前に競合調査→計画の確認を挟みます。
作りたいもの 必須
どんなアプリを、何のために作りたいか。自由に書いてください。
アプリの種類
主な機能・画面
必要な機能や画面を箇条書きで。思いつく範囲でOK。
想定ユーザー・利用場面
誰がどんな場面で使うか。
競合調査で特に知りたいこと
実装前に既存の類似製品・OSS・ライブラリを調査して報告します。特に知りたい観点があれば。
追加したい機能・画面 必須
何を追加・変更したいか。
関連する既存機能・画面
影響しそうな既存部分が分かっていれば。
競合調査の要否
新しいライブラリの選定を伴いそうなら「必要」を推奨。
技術的な制約・希望
使いたい/避けたいライブラリ、Python バージョン、動かす環境など。
